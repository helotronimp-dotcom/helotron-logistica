  1. Clique em "Copiar assinatura" acima
  2. Abra o Gmail → ⚙️ → Ver todas as configurações
  3. Aba Geral → seção "Assinatura" → "Criar nova"
  4. Nome: Helotron
  5. Cole com Ctrl+V no editor de assinatura
  6. Defina como padrão para Novos emails e Respostas
